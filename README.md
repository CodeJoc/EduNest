# EduNest 🎓

EduNest is a full-stack EdTech platform tailored for young learners. It provides a clean, interactive UI where admins can create and manage courses, and students can register, enroll in courses, track progress, and view video lectures.

## ✨ Features

### 🔐 Authentication
- Register/Login as Student or Admin
- Role-based dashboard (Admin vs Student)
  
### 👨‍🏫 Admin Panel
- Create and manage courses
- Upload video lectures
- View number of users

### 🎓 Student Dashboard
- Browse and enroll in courses
- Watch video lectures
- Track course progress
- View and edit profile

### 🌐 Pages
- Landing Page with Home, About, and Login
- Login/Register
- About EduNest
- Course Details with lectures

---

## 🛠️ Tech Stack

### Client
- React.js
- React Router DOM
- Axios
- CSS Modules

### Server
- Node.js
- Express.js
- MongoDB
- JWT Authentication
- Multer (for video uploads)
